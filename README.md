# TravelGuru v2

TravelGuru v2 is an AI-powered travel planning platform built from first principles using modern full-stack engineering and Generative AI technologies.

The application enables users to describe their trip in natural language and receive intelligent travel recommendations, ranked flight and hotel options, destination insights, weather information, maps, and AI-generated itineraries using real-world data sources.

## Tech Stack

### Frontend

* React 19
* TypeScript
* Vite
* Tailwind CSS
* TanStack Query

### Backend

* FastAPI
* Python 3.12
* LangGraph
* LangChain

### AI

* Groq Llama 3.3 70B
* NVIDIA NIM (Fallback)
* Retrieval-Augmented Generation (RAG)

### Database

* Supabase
* pgvector

### Infrastructure

* AWS Lambda
* API Gateway
* AWS Amplify

* CloudWatch
